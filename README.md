### What I Do

- **Kubernetes Platforms** — Design and operate production K8s clusters (AKS, EKS, self-managed) with OPA/Gatekeeper, Pod Security Admission, and CIS hardening
- **GitOps & CI/CD** — GitHub Actions pipelines, ArgoCD App-of-Apps, multi-environment promotion with approval gates
- **Infrastructure as Code** — Terraform (modular, remote state), Bicep, Ansible, Helm chart authoring
- **Observability** — Prometheus, Grafana, Loki, OpenTelemetry — full-stack metrics, logging, and distributed tracing
- **Platform Engineering** — Self-service namespace provisioning, policy-as-code guardrails, developer experience
